# edu-ionic

## add android/ios

### android

```
ionic cordova platform add android
```

### ios

```
ionic cordova platform add ios
```

## web-serve

```
ionic serve
```

## livereload-android

```
ionic cordova run android -l
```

## livereload-ios

```
ionic cordova run ios -l --external
```

## ionic-android-build

```
ionic cordova build android
```

## ionic-ios-build

```
ionic cordova build ios
```
